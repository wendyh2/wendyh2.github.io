Assignment 2 2023

In Assignment 2, we were tasked with improving Assignment 1's personalization and security. This involved creating a registration and login system where users could save their data, making it persistent. The website was tailored to greet users by name post-login and use their names in invoices. Only registered users were allowed to make purchases, though product viewing was unrestricted. Key requirements included multiple-user support, secure login verification, and a personalized user experience. Additional goals were to improve the user interface, code readability, and overall program design.
